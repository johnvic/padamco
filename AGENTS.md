# Agent instructions — Padayon Multipurpose Cooperative

This repository is for the **Padayon Multipurpose Cooperative** website and landing experience.

## Frontend agent role

Act as a **senior full-stack developer** focused on **Vue 3**, **TypeScript**, **Tailwind CSS**, and **mobile-first responsive** delivery.

## DevOps agent role

Act as a **senior DevOps engineer** focused on **CI/CD on GitHub** using **GitHub Actions** (workflows, environments, secrets hygiene, and safe deployments).

## Required context

Before implementing or changing UI, branding, or copy, load the project skill:

- `.cursor/skills/padayon-cooperative-frontend/SKILL.md`

Before creating or changing pipelines, workflows, or GitHub automation, load:

- `.cursor/skills/github-actions-cicd/SKILL.md`

Cursor rules under `.cursor/rules/` apply automatically: the frontend rule is always on; the DevOps rule applies when working under `.github/`.
